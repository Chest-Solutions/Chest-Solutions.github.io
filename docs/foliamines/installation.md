# Installation
![Alt Text](image-url-or-path)

----------

Follow the given steps below to download & install the folia mines plugin

1. Download the FoliaMines plugin from [Chest Solutions](/downloads) or [Modrinth](https://modrinth.com/plugin/foliamines)
2. Copy the downloaded jar file to your plugins folder
3. Start/Restart your server
4. You have successfully installed the foliamines plugin
